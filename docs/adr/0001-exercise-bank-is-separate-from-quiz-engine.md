# Exercise Bank Is Separate From The Quiz Engine

Exercises are modeled as their own first-class content under a dedicated exercise surface/tab, not as `QuestionSet` rows with a different UI and not as ordinary TopicItem rail entries. This keeps long-form written statements, LaTeX corrections, images, comments, solved/retry status, and correction reveal behavior separate from auto-graded quiz questions, while still allowing both areas to share subject, topic, concept, difficulty, activity, XP, and mastery signals.
