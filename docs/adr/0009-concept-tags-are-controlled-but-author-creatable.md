# Concept Tags Are Controlled But Author-Creatable

Concept tags form a controlled vocabulary for filtering, mastery, mistake review, search, and AI retrieval, but content authors can create or propose tags during editing and imports. Proposed tags are usable internally so imports are not blocked, but only approved tags appear in student-facing filters/search; admin tooling must support canonicalization, merge/rename, approval or cleanup workflows, and duplicate detection.
